# 📊 Classification d'Images — Science des Données

Projet académique réalisé en L1 Maths-Informatique à l'Université Paris-Saclay (2026),
dans le cadre du cours de Science des Données (Python).

## Description
Système de classification d'images basé sur l'extraction d'attributs
et la comparaison de plusieurs classificateurs.

### Fonctionnalités
- Extraction d'attributs visuels
- Comparaison de classificateurs selon la méthode VI-ME-RE-BAR
- Analyse des performances

## Technologies
- Langage : Python
- Domaine : Machine Learning / Science des données
